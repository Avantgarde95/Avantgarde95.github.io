## Astro - 아일랜드 아키텍처

설명

- 정적인 HTML 위의 각 UI 컴포넌트
- 아일랜드는 다른 아일랜드와 격리되어 실행

장점

- 대부분 정적 HTML / 필요한 컴포넌트들만 JS 로드 -> 성능 향상
- 아일랜드 간 격리 -> 병렬 로딩 (아일랜드가 다른 아일랜드 차단 X)
- 한 페이지에서 React, Vue 등 여러 프레임워크 혼합 가능

종류

- 서버 아일랜드
- 클라이언트 아일랜드

```astro
<!--HTML, CSS만 사용. JS는 제거됨.-->
<MyComponent />

<!--이제 상호 작용 가능.-->
<MyComponent client:load />

<!--서버 아일랜드.-->
<MyComponent server:defer />
```
